# ISP 2026

PISP provides source-acquisition support for selected 2026 Integrated System Plan material.
`PISP.download_isp2026_assets` downloads selected source assets, and `PISP.download_ISP26_reports` downloads selected report PDFs.

The source-asset downloader does not extract archives or derive trace data.
PISP does not provide an ISP 2026 parser, dataset builder, generated-output contract, validation page, or analysis page.
Downloaded 2026 material should therefore be treated as source material rather than as PISP-generated data.

The [source material](source-material.md) page lists the supported source families.
The [source inventory](source-inventory.md) page separates source downloads from parsed and generated data.

## Using 2026 source material

An ISP 2026 analysis requires release-specific interpretation, data extraction, and validation outside the current PISP workflow.
In particular, a source download does not establish a 2026 schema, a trace-selection rule, or a mapping to the 2024 PISP output model.

The [comparison guide](comparison.md) identifies the crosswalks required for a defensible comparison with ISP 2024 material.
